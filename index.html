What's New

  • New Canteen in school

    • Some new changes are there in our school canteen. First is that the canteen is now divided into 2 section. One is teacher canteen section and the other is children canteen section. Now there are new food items added to it. Example puff, ice-cream, taco, hot-dog, cold drink and many more. There are more tables arranged so that more people can sit comfortably. There is a speaker for theme music. There is a token system started in the canteen. There are 2 vending machine added to it.

  • Online classes

    • Now in the peroid of lockdown, our school has started online classes from a app name google classroom. In this online platform, there inside the google classroom there is a google meet. The function of google classroom is that we can chat with our friends and teachers. And if there is a announcement the teacher has to announce, so the teacher uses google classroom if she want to announce something or to send the code for the meeting. The function of google meet is that first the teacher has to send a code for the meeting in the google classroom and we have to put that code in the google meet. We can't join this meeting BEFORE the teacher joins the meeting. When the student joins the class the google meet marks it present and gives the duration to the teacher that when the student joined the class and when the student left the class. Some new features are added to the google meet but it is optional that is to change our wallpaper. From school, each student has got his own gmail id from school. We can use this school id not like a normal manner, like we use our normal gmail id. We can only use this gmail id in gmail, google classroom and google meet.
